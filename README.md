# Bike'n You — site recréé

Site statique prêt pour GitHub Pages.

## Contenu
- `index.html` : site complet FR/EN
- `assets/` : logo, photos de l'atelier et avatars d'avis
- `privacy.html` et `legal.html`

## Mise en ligne
Téléversez tout le contenu de ce dossier à la racine du dépôt GitHub, puis activez GitHub Pages sur la branche `main`, dossier `/root`.
